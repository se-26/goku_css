Características del Diseño

background-size: cover

La imagen cubre todo el contenedor manteniendo sus proporciones. Puede recortar parte de la imagen si es necesario.

background-position: center

La imagen se centra tanto horizontal como verticalmente dentro del contenedor.

background-repeat: no-repeat

La imagen no se repite, aparece una sola vez en el contenedor.

Capa RGBA semitransparente

Un overlay con rgba(0, 0, 0, 0.6) mejora la legibilidad del texto sobre la imagen.

Variaciones de Configuración

Configuración Original

background-size: cover;

Imagen completa cubriendo el área

Overlay Más Oscuro

rgba(0, 0, 0, 0.75)

Mayor opacidad para más contraste

Overlay de Color

rgba(231, 76, 60, 0.5)

Overlay rojo semitransparente
