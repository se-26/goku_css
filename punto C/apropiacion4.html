Características Técnicas Implementadas

Gradiente Lineal

Cada banner utiliza un gradiente como fondo base para crear profundidad visual

background: linear-gradient(135deg, color1, color2);

Imagen Decorativa

Imagen de Goku posicionada estratégicamente con background-position

background-image: url('goku.jpg');

Texto Contrastante

Colores de texto cuidadosamente seleccionados para máxima legibilidad

color: #ffffff; text-shadow: 2px 2px 4px rgba(0,0,0,0.5);

Efectos Interactivos

Transformaciones y sombras al hacer hover para feedback visual

transform: translateY(-5px); transition: all 0.3s;
